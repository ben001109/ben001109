## 🧭 我重視的事

- 先把問題縮成一個清楚的使用流程，再談擴張。
- 偏好小而完整、能被實際使用的工具。
- 保留實驗性，但不把「還沒驗證」包裝成完成品。
- 讓技術服務人的創作、學習與日常工作。
